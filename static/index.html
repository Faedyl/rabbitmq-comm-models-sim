Model Request-Response

Cara Kerja: Client mengirim request ke Server, lalu Server memproses dan mengirim response kembali. Model ini bersifat sinkron — client menunggu sampai response diterima sebelum melanjutkan. Contoh dunia nyata: HTTP API, REST services, database queries.

Model Publish-Subscribe

Cara Kerja: Publisher mengirim pesan ke sebuah topic/channel. Semua Subscriber yang terdaftar pada topic tersebut akan menerima notifikasi secara asinkron. Publisher tidak perlu tahu siapa saja subscriber-nya (loose coupling). Contoh dunia nyata: MQTT (IoT), Apache Kafka, Redis Pub/Sub, event bus.

Model Remote Procedure Call (RPC)

Cara Kerja: Caller memanggil prosedur/fungsi pada mesin remote seolah-olah memanggil fungsi lokal. Proses melibatkan: marshalling parameter di client stub → pengiriman via jaringan → unmarshalling di server stub → eksekusi → return hasil melalui jalur yang sama. Contoh dunia nyata: gRPC, XML-RPC, Java RMI.
